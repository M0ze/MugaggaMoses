# 🚀 Mugagga Moses' Terminal Portfolio

**A sleek, responsive, retro-futuristic developer portfolio.**

[![Live Demo](https://img.shields.io/badge/Live_Demo-Online-39ff14?style=flat&logo=githubpages&logoColor=white)](https://mugaggamoses.github.io/terminal-portfolio/)

This portfolio is designed to look like a high-end digital terminal. It provides a seamless, immersive experience across all devices, from desktop monitors to mobile phones.

## ✨ Features

- **Modern Retro Design**: Clean terminal aesthetics with subtle CRT scanline effects.
- **Fully Responsive**: Optimized for all screen sizes, ensuring readability and accessibility.
- **Multi-Page Architecture**: Dedicated sections for About, Skills, Projects, Experience, and Contact.
- **Fast & Lightweight**: Built with Vanilla HTML and CSS for maximum performance and zero dependencies.
- **Professional Journey**: Interactive timeline and skill bars to showcase expertise.

---

## 🧬 Tech Stack

- **Frontend**: HTML5, CSS3 (Custom Variables, Media Queries)
- **Typography**: 'VT323' Monospace (Google Fonts)
- **Deployment**: GitHub Pages via GitHub Actions

## 🚀 Development & Deployment

### Local Development
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No build step required!

### Deployment
This project is configured for **GitHub Pages**. Any changes pushed to the `main` branch will automatically trigger the deployment workflow defined in `.github/workflows/static.yml`.
